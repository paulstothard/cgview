Creating a Map
  • Use the CGView application - The CGView application can be used to generate static images (in PNG, JPG, or SVG format), or to generate a linked set of map images that permit genome navigation and linking to external resources. Once the linked maps are generated they can be served as standard web content. The CGView application is designed to be run from the command line, so that it can be integrated into sequence analysis pipelines. An included script called cgview_xml_builder.pl can be used to generate the XML input needed for CGView, from genome sequence files in a variety of formats. See the CGView GitHub repository for more details on cgview_xml_builder.pl.
  • Use the CGView API - The CGView API can be used by other Java applications to create maps.
  • Use the CGView Server - The CGView Server can be used to create maps online.
